# ProgressManager

休日の予定を楽しく管理するシンプルな進捗アプリです。

## 🌟 概要
土曜日のデートプランをテーマに、  
予定の管理・進捗の可視化・ランダム提案を行えるWebアプリです。

## 🔧 機能
- 予定のチェック（完了 / 未完了）
- 進捗バー表示（％で可視化）
- 進捗に応じたステータスメッセージ
- ランダムで次の予定を提案
- タスクごとのアドバイス表示
- 完了時のランダム演出（✨・❤️・🎉）
- localStorageによるデータ保存

## 🎮 こだわりポイント
- ボタンを押したくなるようなUI設計
- ランダム要素による「飽きない体験」
- 軽い演出でゲーム感をプラス

## 🚀 使用技術
- HTML
- CSS
- JavaScript（Vanilla JS）
